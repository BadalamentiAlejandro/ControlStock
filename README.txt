
In this project I created a simple GUI with Swing that is connected to a MySQL data base. The user is able to add, modify and delete information shown on a JTable validated with PreparedStatement.

En este proyecto he creado una GUI sencilla con Swing que se conecta a una base de datos MySQL. El usuario tiene la posibilidad de agregar, modificar y borrar informacion que se muestra en una JTable mediante validaciones con PreparedStatement.
